# Contact Support

## EvalAI hosted platform (eval.ai)

- Email: [team@eval.ai](mailto:team@eval.ai)
- Challenge-specific questions: contact addresses listed on the challenge page
- Bugs and feature requests: [GitHub Issues](https://github.com/Cloud-CV/EvalAI/issues/new)

## Before contacting support

Include:

- Challenge URL or challenge ID
- Phase name or codename
- Submission ID (if applicable)
- Steps to reproduce
- Screenshots or redacted logs

## Self-hosted EvalAI

Operators maintain their own instances. Use your internal support channel or the open-source community via GitHub for platform bugs.
